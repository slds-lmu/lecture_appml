\begin{vbframe}{Model Averaging}
Combine predictions of multiple (different / unique) models to make a final prediction

Leverages strengths and diversity of different models to improve overall performance and robustness
\end{vbframe}

% ------------------------------------------------------------------------------

\begin{vbframe}{Steps}

\begin{itemize}
\item Base model training
\item Collect and combine predictions
\end{itemize}
\end{vbframe}

% ------------------------------------------------------------------------------

Note on post-hoc ensembling
\end{vbframe}

\begin{vbframe}{Collect and Combine Predictions}
  Regression vs. Classification
  Weighted averaging vs. averaging (selection) Note: Should we even bother?

  \begin{itemize}
    \item Combine predictions of base models by taking a weighted average
    \item Weights represent importance of a base model's prediction
    \item Can also be tuned
  \end{itemize}

  \begin{itemize}
    \item Combine predictions by simply averaging the predictions of base models
    \item Potentially also only selecting a subset of base models
    \item Selecting base models is equal to weighted averaging with weights only 0 or 1
   \end{itemize}
\end{vbframe}
